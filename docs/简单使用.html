简单使用

简单使用

开发一个简单页面组件,比如表单组件：myForm.vue

  1. 在components文件夹中新建一个myForm.vue文件，里面输入html模版和script。templat里面只允许有一层节点

    <template>
   <Form ref="formInline" :model="formInline" :rules="ruleInline" inline>
       <FormItem prop="user">
           <Input type="text" v-model="formInline.user" placeholder="Username">
               <Icon type="ios-person-outline" slot="prepend"></Icon>
           </Input>
       </FormItem>
       <FormItem prop="password">
           <Input type="password" v-model="formInline.password" placeholder="Password">
               <Icon type="ios-locked-outline" slot="prepend"></Icon>
           </Input>
       </FormItem>
       <FormItem>
           <Button type="primary" @click="handleSubmit('formInline')">Signin</Button>
       </FormItem>
   </Form>
</template>
<script>
   export default {
       data () {
           return {
               formInline: {
                   user: '',
                   password: ''
               },
               ruleInline: {
                   user: [
                       { required: true, message: 'Please fill in the user name', trigger: 'blur' }
                   ],
                   password: [
                       { required: true, message: 'Please fill in the password.', trigger: 'blur' },
                       { type: 'string', min: 6, message: 'The password length cannot be less than 6 bits', trigger: 'blur' }
                   ]
               }
           }
       },
       methods: {
           handleSubmit(name) {
               this.$refs[name].validate((valid) => {
                   if (valid) {
                       this.$Message.success('Success!');
                   } else {
                       this.$Message.error('Fail!');
                   }
               })
           }
       }
   }
</script>

2.有了组件，但是希望这个组件能够有个样式，比如想要有个好看一点的盒子包裹住组件，adminlte中有各种盒子可以选择。

我们选择“主要盒子”作为演示。

首先，盒子的主要结构如下：

```
<!--  需要给div加个类名.box -->
<div class="box">
    <!-- 盒子的头部.box-header,如果头部需要添加边框可以带上.with-border -->
    <div class="box-header with-border">
        <!-- 盒子的标题信息 -->
        <h3 class="box-title">Default Box Example</h3>
        <!-- 盒子的工具，需要加上.box-tools,如果想把工具展示在盒子右侧需要加上能右移的类.pull-right -->
        <div class="box-tools pull-right">
            <!-- 按钮Buttons, 标签labels, 还有其他东西都可以在这里写! -->
            <!-- 比如想要在工具栏中添加一个标签.label -->
            <span class="label label-primary">Label</span>
        </div>
    </div>
    <!-- 盒子的主体部分.box-body -->
    <div class="box-body">
        The body of the box
    </div>
    <!-- 盒子的底部 -->
    <div class="box-footer">
        The footer of the box
    </div>
</div>
```

根据盒子的结构，我们可以把组件放在body部分，然后给组件添加一个title在header部分，把按钮相关的元素放在盒子footer，整理之后html代码如下:

```
<div class="box box-primary">
    <div class="box-header with-border">
        <h3 class="box-title">表单</h3>
    </div>
    <Form ref="formInline" :model="formInline" :rules="ruleInline" inline>
        <div class="box-body">
            <FormItem prop="user">
            <Input type="text" v-model="formInline.user" placeholder="Username">
                <Icon type="ios-person-outline" slot="prepend"></Icon>
            </Input>
        </FormItem>
        <FormItem prop="password">
            <Input type="password" v-model="formInline.password" placeholder="Password">
                <Icon type="ios-locked-outline" slot="prepend"></Icon>
            </Input>
        </FormItem>
        </div>
        <div class="box-footer">
            <button type="submit" class="btn btn-primary pull-right">Sign in</button>
        </div>
    </Form>
</div>
```

组件组合成页面

页面的开发和组件基本一样，由不同的组件组合成一个复杂的大组件。这个大的组件可以用来配置路由对应的页面，文件位置在views文件夹中。

组成页面之后，就需要在浏览器中展示出来。但是页面需要通过路由来访问才可以展示，下面就需要配置路由来打开页面了。比如通过上面的myForm组件，和其他组件组件成一个页面。

在component文件夹下新建一个test/test.vue文件

<template>
    <div>
        <myForm/>
        <myForm1/>
        <myForm2/>
    </div>
</template>
<script>
    import myForm from '../../components/myForm.vue'；//如果需要引用myForm组件
    import myForm1 from '../../components/myForm1.vue'；//如果需要引用myForm1组件
    import myForm2 from '../../components/myForm2.vue'；//如果需要引用myForm2组件
    export default {
    }
</script>

这样就完成一个简单的页面。

配置路由

在router文件夹中，menu.js主要配置左侧菜单路由，router.js主要配置其他跳转的路由。下面以router.js文件为例。

  1. 引入页面组件,例如配置上面的test组件到路由
    import Test from '../views/test/Test.vue';

  2. 配置路由

    const Routers = [{
       path: '/test',  //路由跳转hash路径
       //redirect: '/demand',  //重定向路径(可选)
       component: Test,   //路由映射的组件
       name: '测试页面',   //路由名称
       //hidden: true,   //如果是在左侧菜单栏，可以通过这个属性控制菜单显示隐藏
       meta: {   //其他配置
           id: 1,    //权限id
           icon: 'fa-dashboard',   //菜单图标
       },
   },

]

这时，就可以通过路由或者左侧菜单访问页面了。
