# 简单使用

### 开发一个简单页面组件,比如Button

  1. 在components文件夹中新建一个Button.vue文件，里面输入html模版。``templat里面只允许有一层节点``

    ```
    <template>
        <div>
            /**使用引进来的组件**/
            <Top/>
            <button>按钮<button>
        </div>
    </template>

    ```
  2. 再输入script 部分

    ```
    <script>
    import Top from '../components/top.vue'；//如果需要引用自定义组件，我们import进来
    export default {
        // 这里写你的代码，外面要包起来。
        //export 用来导出模块，Vue 的单文件组件通常需要导出一个对象，
        //这个对象是 Vue 实例的选项对象，以便于在其它地方可以使用 import 引入。
    }
    </script>
    ```

有了组件，就可以用组件开发页面了。

### 组件组合成页面

页面的开发和组件基本一样，由不同的组件组合成一个复杂的大组件。这个大的组件可以用来配置路由对应的页面，文件位置在views文件夹中。

组成页面之后，就需要在浏览器中展示出来。但是页面需要通过路由来访问才可以展示，下面就需要配置路由来打开页面了。

### 配置路由

在router文件夹中，menu.js主要配置左侧菜单路由，router.js主要配置其他跳转的路由。下面以router.js文件为例。

  1. 引入页面组件,例如
  ```javascript
    import Demand from '../views/demand/Demand.vue';
    import error403 from '../views/errorpage/403.vue';
  ```
  2. 配置路由
  ```javascript
    const Routers = [{
            path: '/',  //路由跳转hash路径
            redirect: '/demand',  //重定向路径
            name: '首页',   //路由名称
            meta: {   //其他配置
                id: 1,    //权限id
                icon: 'fa-dashboard',   //菜单图标
            },
        },
        {
            path: '/403',
            component: error403,   //路由映射的组件
            hidden: true,   //如果是在左侧菜单栏，可以通过这个属性控制菜单显示隐藏
            meta: {
                id: 2
            }
        },

    ]
  ```

这时，就可以通过路由或者左侧菜单访问页面了。
